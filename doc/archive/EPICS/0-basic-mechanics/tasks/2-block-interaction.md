# TASK: Взаимодействие с блоками (Block Place/Break)

**Layer**: 1
**Статус**: Draft
**Эпик**: 0-basic-mechanics

## Affected Services

| Service | Role | R/W |
|---------|------|-----|
| **SimulationCore** | Primary — проверяет наличие предмета, расходует блоки | Read/Write |
| **ChunkStore** | Block storage — записывает/читает блоки | Write/Read |
| **Gateway** | Relay — forwards PlayerAction между клиентом и сервисами | Relay |
| **GameClient** | Consumer — отправляет PlayerAction, получает InventoryUpdate | Read/Write |

---

## Описание

Игрок ставит и ломает блоки в мире, используя выбранный слот хотбара. Механика базовая — без инструментов, прочности и дропа.

---

## Требования

### Установка блоков
- Игрок выбирает слот хотбара с блоком
- ЛКМ по миру → блок ставится в указанную позицию
- Блок расходуется из инвентаря (count--, если 0 — слот пустеет)

### Ломка блоков
- Зажать ЛКМ на блоке → блок разрушается
- Сломанный блок сразу попадает в инвентарь (без анимации, без дропа на землю)
- Если инвентарь полон — блок теряется (MVP)

### Flow
```
Client: PlayerAction{action: PLACE/BREAK, x, y, z, block_id, selected_slot}
    → Gateway
    → SimulationCore: валидация, обновление инвентаря
    → ChunkStore: SetBlock
    → SimulationCore: InventoryUpdate
    → Gateway
    → Client: обновлённый инвентарь
```

### Упрощения MVP
| Упрощение | Почему |
|-----------|--------|
| Нет инструментов | Любой предмет/рука ломает блок с одинаковой скоростью |
| Нет прочности | Предметы не расходуются (кроме блоков при постройке) |
| Нет дропа | Блок сразу в инвентарь — не нужна физика |
| Полное доверие | Клиент сообщает действие, сервер верит |

---

## Protocol

```flatbuffers
table PlayerAction {
    player_id: uint64;
    action: PlayerActionType;  // PLACE, BREAK
    x: uint32; y: uint32; z: uint32;
    block_id: uint16;
    selected_slot: uint8;      // выбранный слот хотбара
}
```

## Acceptance Criteria

#### Сценарий: Поставить блок
1. Игрок выбирает слот с блоком земли (item_id=3)
2. Наводит курсор на соседний блок, нажимает ЛКМ
3. Блок земли появляется в мире
4. Счётчик земли в хотбаре уменьшается на 1

#### Сценарий: Сломать блок
1. Игрок наводит курсор на блок камня
2. Зажимает ЛКМ
3. Блок исчезает из мира
4. Камень появляется в инвентаре
5. Никаких выпавших предметов на земле нет

#### Сценарий: Блок из пустого слота
1. У игрока выбран пустой слот (item_id=0)
2. Игрок нажимает ЛКМ
3. Блок не ставится (нет предмета)
4. Игрок всё ещё может ломать блоки

#### Сценарий: Полный инвентарь
1. Все 45 слотов инвентаря заняты
2. Игрок ломает блок
3. Блок исчезает из мира
4. В инвентарь блок НЕ добавляется (теряется)
